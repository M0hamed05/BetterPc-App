# BetterPC 

**Imagine a tool that combines every way of optimizing your PC into a single, seamless dashboard. That’s BetterPC.**

### ⚡ Speed & Performance Optimization
* **Startup Manager Shortcut:** Direct link to disable resource-heavy startup applications.
* **Background Apps Management:** Quick access to toggle off Windows apps running silently in the background.
* **Visual Effects Tweaker:** Guided shortcut to the native 'Performance Options' to disable heavy UI animations while preserving fonts and thumbnails.
* **Safe Services Guide:** Direct access to `msconfig` with instructional popups ensuring users hide essential Microsoft services before configuring.

### 💾 Storage Management
* **Temp Files Cleaner:** Dedicated UI to scan and purge safe temporary files, prefetch data, and system logs.
* **Disk Cleanup Utility:** Launches legacy Windows Disk Cleanup to target deep Windows update leftovers.
* **Hibernation Toggle:** Instantly enables or disables Hibernation via Registry check, allowing users to free up gigabytes of space by managing `hiberfil.sys`.
* **Modern Storage Settings Link:** Quick path to Windows 'Storage Sense' for automated, hands-off drive management.
